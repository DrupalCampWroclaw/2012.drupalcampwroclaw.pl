Szybko, szybciej, Cherokee... przyspiesz swojego Drupala.

Opublikował/a artur.smolarek o godz śr., 2012-01-11 22:58
Prowadzący/ca: 
Artur Smolarek, Jędrzej Nowak

Tematem prezentacji będzie wprowadzenie do Cherokee Web Server - omówienie idei jego działania, Cherokee Admin-a, przedstawienie Cherokee Market i zasady działania front line cache.
Przeprowadzimy również porównanie Cherokee Web Server z innymi dostępnymi na rynku rozwiązaniami (głównie Apache) oraz pokażemy jak w kilku krokach zainstalować Drupala przy pomocy Cherokee Admin-a.
Prezentacja zostanie poprowadzona przez firmę Codernity, oficjalnego przedstawiciela Cherokee Project na Polskę.
